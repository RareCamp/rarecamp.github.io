About RareCamp


RareCamp, a Linux Foundation project, provides open governance for the software and scientific community to collaborate and create the software tools to aid in the creation of treatments for rare diseases.


RareCamp was started as a collaboration between OpenTreatments Foundation and Linux Foundation to build a software platform to create gene therapies for rare diseases. This open source project powers the OpenTreamtments software platform, which is used by patient-led organizations to find treatments
